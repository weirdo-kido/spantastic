# spantastic

A visual experiment that turns heavenly chaos into terminal-friendly pixels. No Ray Tracing, no DLSS, just pure ASCII aesthetics.

---

### The Aesthetic
- Source Image: [**andsproject | pixabay**](https://pixabay.com/vectors/girl-hope-dream-anime-9599953/)
- Engine: Powered by [JasperBernaers](https://jasperbernaers.com/ASCII-generator/) Image-to-ASCII magic.

### Technical "Flex"
- **0ms JavaScript:** It’s so fast, because script is not found.
- **25k Spans:** Handcrafted by the CPU gods.
- **Works on my machine:** The only benchmark that actually matters in 2026.

---

### Usage
1. Open it.
2. Stare at 25k span.
3. Realize you didn't need a 50-series card for this.
